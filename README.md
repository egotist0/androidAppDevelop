## chapter 6: todo list
* 为Todo List的场景建立一个数据库，完成数据库表的设计和创建
* 进入主页后，以列表的形式展现出数据库中所有的数据
* 点击“+”后，跳转到一个新页面，输入任意类容，把类容插入数据库，返回主页并更新UI
* 点击任意一条checkbox可以把该条note设置为已完成，并更新数据库及UI
* 点击任意一个“x”，可以删除该条note，并更新数据库及UI
